Práctica 2 - Fundamentos de Bases de Datos 2018-2

La práctica se realizó en Netbeans por lo que se sugiere usarlo para ejecutar la práctica.

Pasos en Netbeans:

	1.- Abrir Netbeans
	2.- Abrir Proyecto
	3.- F6 para ejecutar el proyecto

Pasos en terminal.

	1.- Entrar desde la terminal al directorio de la práctica $ cd ./../Practica2/
	2.- Ejecutar el comando $ ant compile
	3.- Entrar al directorio ./Practica2/build/classes/
	4.- Ejecutar la practica con el comando $ java Practica2
